## Код

---

Код, встроенный в строку: `console.log('Hello, there')`

Блок кода, написанный на языке Python:

```Python
from random import choice

class CrazyAnimal:
  def __init__(self, name: str, reasons: List[str]):
    self.name = name
    self.reasons = ['боли', 'счастья', 'гнева'] + reasons

  def make_sound(self, count: int):
    for _ in range(count):
      print(f'{self.name} кричит от {choice(self.reasons)}')
```

## Выделение слов

---

Далее рассмотрим возможности форматирование текста посредством Markdown тегов.
Начнем с того, чтобы сделать текст курсивным и указать акцентное слово,
используется пара нижних подчеркиваний: _важный текст_

Чтобы сделать текст жирным, используем по два символа звездочки слева и справа:
**очень важный текст**

## Соотношение тегов HTML к тегам Markdown:

---

- `<em>example</em>` === \_example\_
- `<strong>example</strong>` === \*\*example\*\*
