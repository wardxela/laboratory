# Полезные фишки на CSS

Полезные паттерны для написания стилей

## Центрирование с `postition: absolute`

Думаю всем знаком способ центрирования элемента, спозиционированного абсолютно относительно родителя.
Трюк заключается в том, чтобы сдвинуть элемент на 50% вниз и на 50% вправо, ну а так как координаты считаются
от левого верхнего угла, с помощью `transform: translate(-50%, -50%)` мы располагали элемент ровно по центру.

Но что, если я скажу, что есть способ проще.

Если расположить элемент все так же абсолютно по отношению к родителю, а затем с помощью свойства `inset`
спозиционировать элемент таким образом, чтобы он прикрепился к краям родителя, то применив `margin: auto`,
подобно тому, как это работает в `grid` и `flex` контейнерах, элемент окажется ровно в центре родителя.

**Примечание:** дочерний элемент должен иметь ограничения, заданные по высоте и ширине, иначе он займет
все имеющееся пространство родителя.

<CodeSample>
  <CodeSample.Code>
    ##### HTML

    ```html
    <div class="parent">
      <div class="child"></div>
    </div>
    ```

    ##### CSS

    ```css
    .parent {
      position: relative;
    }

    .child {
      width: 40px;
      height: 40px;

      /* Располагает элемент ровно в центре родителя */
      position: absolute;
      inset: 0;
      margin: auto;
    }
    ```

  </CodeSample.Code>
  <CodeSample.Result>
    <div style={{
      position: 'relative',
      border: '4px dashed #222',
      height: '100%',
    }}>
      <div style={{
        position: 'absolute',
        inset: 0,
        margin: 'auto',
        height: '40px',
        width: '40px',
        background: 'rgb(59 130 246)'
      }}></div>
    </div>
  </CodeSample.Result>
</CodeSample>
